# BookShelf API

BookShelf API is a simple REST API built with Go and PostgreSQL.

The project allows a single user to log books and store personal reading information, such as reviews, price, and read date.  
It is inspired by Letterboxd, but focused on books.

## Purpose

The main goals of this project are:
- Practice backend development using Go
- Learn how to design and implement REST APIs
- Apply database modeling concepts
- Improve my technical English while coding

## Features (v1)

- Create a book
- Get all books
- Get a book by ID
- Update a book
- Delete a book

## Tech Stack

- Go
- PostgreSQL
- net/http
- database/sql

## Project Status

This project is currently under development.  
It starts as a single-user application and is designed to support multiple users in the future.
